# onakawash

语言：[English](./README.md) | [中文](./README.zh-CN.md)

## 项目介绍

onakawash 是一个面向日语初学者的五十音学习网站。

这个项目目前主要包含平假名、片假名、音频播放、多语言首页，以及前端和 Spring Boot 后端的数据连接。

项目仍在开发中。

## 当前功能

* 自定义视觉设计首页
* 中文 / 英文 / 韩文语言切换
* 平假名学习页面
* 片假名学习页面
* 点击假名播放音频
* 点击新假名时停止上一个音频
* 前端连接 Spring Boot 后端
* 数据按照清音、浊音/半浊音、拗音分类
